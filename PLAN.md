# GameFi TON Mining Simulator — План разработки

> Telegram Mini App · Блокчейн TON · Жанр: Экономический менеджер / Idle

---

## Обзор проекта

Симулятор виртуального майнинга с реальными выплатами в TON. Игрок строит ферму из видеокарт и ASIC-майнеров, управляет износом оборудования, торгует на вторичном рынке и зарабатывает долю от глобального пула наград пропорционально своей мощности.

**Стек:** React (TMA) · Node.js + PostgreSQL + Redis · FunC / Tact (TON)  
**Команда:** 3–4 человека  
**Общий срок:** ~15–17 недель

## Валютная система

- **TON (Hard)** — покупка оборудования, расширений, вывод прибыли
- **IGC (Soft / In-Game Coin)** — добывается майнингом, тратится на электричество, ремонт, верстаки

## Ключевые формулы

### Добыча (Pool Mining)
```
P = (H_user / H_total) × R_epoch
```

### Реферальный хешрейт (Управляющая компания)
```
H_total_user = H_base + Σ(H_L1 × 0.05) + Σ(H_L2 × 0.02)
```

### Износ оборудования
```
W = W_base × K_temp × K_load
```

### Шанс критической поломки
```
P_fail = ((100 − Health) / 100)³
```

### Цена оборудования в магазине
```
Price(t) = Price_base × (1 − d)^t × M_profit
```

### Стоимость восстановления (Refurbish)
```
Cost_refurbish = (100 − Health) × BaseCost × Multiplier_tier
```

### Solo майнинг — шанс найти блок
```
P_win = H_user / H_total
```

## Anti-Fraud чеклист

- [ ] Валидация `initData` от Telegram на каждый запрос
- [ ] Все расчёты — только backend (клиент не доверяем)
- [ ] Hold 24–48ч перед первым выводом TON
- [ ] Лимит вывода: не более 1% пула в сутки (контракт)
- [ ] Ценовой коридор на маркетплейсе: ±20% от рыночной цены
- [ ] Слепой стакан в маркетплейсе (нет целевых переводов)
- [ ] Rate limit на Tap-to-Cool (макс. N кликов в секунду)
